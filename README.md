# 📖 Overview

**LoveLens-AI** is an intelligent relationship analysis application that evaluates compatibility using AI-driven logic and interactive questionnaires.

The application collects responses from users, analyzes multiple relationship dimensions, generates compatibility insights, and provides a detailed report through a modern Streamlit interface.

This project demonstrates practical skills in:

* Machine Learning
* Python Development
* Streamlit Application Development
* Data Processing
* User Experience Design
* AI-based Decision Systems

# 🚀 Installation

## 1. Clone the Repository

```bash
git clone https://github.com/mangesh-rathod-ai/LoveLens-AI.git
```

---

## 2. Navigate to the Project

```bash
cd LoveLens-AI
```

---

## 3. Create Virtual Environment

```bash
python -m venv venv
```

---

## 4. Activate Virtual Environment

### Windows

```bash
venv\Scripts\activate
```

### Linux / macOS

```bash
source venv/bin/activate
```

---

## 5. Install Dependencies

```bash
pip install -r requirements.txt
```

---

## 6. Run the Application

```bash
streamlit run app.py
```
